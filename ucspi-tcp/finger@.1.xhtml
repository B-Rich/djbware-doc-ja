finger@(1) - ucspi-tcp

名前

finger@ - リモートホストからユーザーの情報を取得します。

書式

finger@ [host [user]]

解説

finger@ は、 ホスト host の TCP ポート 79 (finger) に接続し、 ユーザー名 user を (末尾に CR (\015) を付加して) 送信し、 データを受け取って表示します。 行末の CR は削除し、 表示できない文字は見える形式に変換します。

コンピューターは TCP ポート 79 への接続に対し、 ユーザー user に関する情報を応答します。

user を指定しないと、 finger@ は host に空白行を送信します。 コンピューターはそのような問い合わせに対し、 ログインしているすべてのユーザーの情報を応答します。

host を指定しないと、 finger@ はローカルホストに接続します。
