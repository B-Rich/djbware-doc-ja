http@(1) - ucspi-tcp

名前

http@ - リモートホストから web ページを取得します。

書式

http@ [host [page [port]]]

解説

http@ は、 ホスト host の TCP ポート port に接続し、 次のようなデータを送信します。 

GET /page HTTP/1.0
Host: host

そして応答の内容から行末の CR (\015) を削除して表示します。

port を指定しないと、 http@ は TCP ポート 80 (HTTP) を使用します。

page を指定しないと、 http@ は host に / を要求します。

host を指定しないと、 http@ はローカルホストに接続します。
