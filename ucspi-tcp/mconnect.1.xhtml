mconnect(1) - ucspi-tcp

名前

mconnect - リモート SMTP サーバーへ接続します。

書式

mconnect [host [port]]

解説

mconnect は、 ホスト host の TCP ポート port に接続します。 そして標準入力から読み込んだ行に CR (\015) を付加して host に送信し、 一方では host から受信したデータを表示します。

port を指定しないと、 mconnect は TCP ポート 25 (SMTP) を使用します。

host を指定しないと、 mconnect はローカルホストに接続します。
