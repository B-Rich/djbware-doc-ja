date@(1) - ucspi-tcp

名前

date@ - リモートホスト上の日時を表示します。

書式

date@ [host]

解説

date@ は、 ホスト host の TCP ポート 13 (daytime) に接続し、 データを受け取って表示します。 行末の CR (\015) は削除し、 表示できない文字は見える形式に変換します。

host を指定しないと、 date@ はローカルホストに接続します。

コンピューターは TCP ポート 13 への接続に対し、 人が読める形式で日時を応答します。 例えば tcpserver(1) を利用して、 次のようなコマンドで実現することができます。 

tcpserver -RHl0 0 13 date
