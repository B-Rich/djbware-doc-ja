softlimit(8) - daemontools

名前

softlimit - 指定されたリソース制限のもとでプログラムを起動します。

書式

softlimit [options] program [arg ...]

解説

softlimit は、 プロセスが利用できるリソースのソフト制限 (soft limits) をオプション options に従い設定し、 プログラム program を引数 arg とともに起動します。

オプション

各オプションにはリソースの制限値 n を指定できますが、 数値の代わりにイコール「=」を指定すると、 ソフト制限はハード制限 (hard limits) と同じ値に設定されます。

メモリ使用量を制御するオプション

-m n
-d n -s n -l n -a n と指定した場合と同じです。
-d n
データセグメントをプロセスあたり n バイト以内に制限します。
-s n
スタックセグメントをプロセスあたり n バイト以内に制限します。
-l n
ロックできる物理ページをプロセスあたり n バイト以内に制限します。 このオプションは、 OS によっては効果がありません。
-a n
すべてのセグメントの合計をプロセスあたり n バイト以内に制限します。 このオプションは、 OS によっては効果がありません。
-o n
オープンできるファイル記述子の数をプロセスあたり n 個以下に制限します。 このオプションは、 OS によっては効果がありません。
-p n
同時に実行できるプロセスの数をユーザー ID あたり n 個以下に制限します。

ファイルのサイズを制御するオプション

-f n
出力ファイルのサイズを n バイト以下に制限します。
-c n
core ファイルのサイズを n バイト以下に制限します。

優先度オプション

-r n
常駐セットのサイズを n バイト以下に制限します。 物理メモリが一杯でないときは、 この制限は課せられません。
-t n
CPU 時間を n 秒以下に制限します。 プロセスが XCPU シグナルを受け取ってから n 秒経過するまでの間を除き、 この制限は課せられません。
