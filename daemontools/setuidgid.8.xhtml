setuidgid(8) - daemontools

名前

setuidgid - ユーザーアカウントのユーザー ID と グループ ID のもとでプログラムを起動します。

書式

setuidgid user program [arg ...]

解説

setuidgid は、 プロセスのユーザー ID とグループ ID を UNIX ユーザーアカウント user のユーザー ID と(プライマリ)グループ ID に設定し、 追加グループ権を外した後、 プログラム program を引数 arg とともに起動します。

setuidgid は、root 以外のユーザーが実行することはできません。

終了コード

次の場合、111 を返します。

  • ユーザーアカウント user が存在しない
  • setgid(2), setuid(2) に失敗
  • プログラム program の起動に失敗

それ以外の場合、 program の終了コードを返します。
