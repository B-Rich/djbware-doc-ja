envuidgid(8) - daemontools

名前

envuidgid - ユーザーアカウントのユーザー ID と グループ ID を環境変数に設定し、 プログラムを起動します。

書式

envuidgid user program [arg ...]

解説

envuidgid は、 UNIX ユーザーアカウント user のユーザー ID を環境変数 UID に、 (プライマリ)グループ ID を環境変数 GID に設定した後、 プログラム program を引数 arg とともに起動します。

このプログラムは、 tcpserver(1) の -U オプションに利用できます。

終了コード

次の場合、111 を返します。

  • ユーザーアカウント user が存在しない
  • 環境変数を設定するためのメモリ確保に失敗
  • プログラム program の起動に失敗

それ以外の場合、 program の終了コードを返します。
